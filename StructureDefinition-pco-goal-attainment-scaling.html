Extension: Goal Attainment Scaling

Official URL: http://hl7.org/fhir/us/pco/StructureDefinition/pco-goal-attainment-scaling Version: 0.2.0
Draft as of 2025-01-09 Computable Name: PCOGoalAttainmentScaling

Goal Attainment Scaling (GAS) is an extension used to specify five levels of attainment for a person-centered goal.

Context of Use

This extension may be used on the following element(s):

  • Element ID Goal

Usage info

Usage:

  • Use this Extension: Person-Centered GAS Goal
  • Examples for this Extension: Goal/pcoGoalWalkDogExample
